# Tender Submission Management System

A static React + Vite tender checklist management app for GitHub Pages.

## Features

- Mock login gate
- Multiple tender projects
- LocalStorage persistence
- Company logo upload
- Checklist management
- Search, filter and sort
- Bulk status update
- Dashboard analytics
- CSV export
- Print / Save as PDF
- GitHub Pages ready
